# 🧠 Quantum Neural Network Visualization

An interactive **3D Neural Network Visualization** built with **Three.js** and custom GLSSL shaders. It renders animated neural structures with glowing nodes, dynamic connections, multiple formations, and interactive energy pulses.

## ✨ Features

- Interactive 3D neural network visualization
- Three built-in color themes
- Click to create animated energy pulses
- Multiple procedural network formations
- Adjustable network density
- Animated starfield background
- Glassmorphism user interface
- Bloom and glow effects
- Orbit camera controls
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Three.js
- GLSL Shaders

## 🚀 Running the Project

Because the project uses JavaScript modules, open it with a local server.

### VS Code
1. Install the Live Server extension.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

### Python

```bash
python -m http.server
```

Open:

```text
http://localhost:8000
```

## 🎮 Controls

| Control | Action |
|---------|--------|
| Left Mouse Drag | Rotate camera |
| Mouse Wheel | Zoom |
| Click | Create an energy pulse |
| Morph | Change network formation |
| Freeze | Pause or resume animation |
| Reset | Reset camera |
| Theme Buttons | Change colors |
| Density Slider | Adjust network density |
